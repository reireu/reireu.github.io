裁判航跡

プロジェクトについて

無限にある重要判例について、国ごとの関わりをイメージの補助となる公開サイト

背景・課題

法は、不文律が多く、あるものも基本的には文章単位が多く、専門的に学んでいる人でも覚えきれないものが無数にある。

主要機能

検索機能

100%完璧覚えていなくても、複数の方法（キーワード、ざっくりとした年数、関わっている国）などから検索できます。

勝訴、敗訴国のレンダリング

勝訴と敗訴に色わけして、国が塗りつぶされているので、視覚的にどのような関係か一目でわかります。

自動再生、自動勝訴国フォーカス

勝訴国に自動的に地球儀が回るほか、2秒ごとに自動で切り替えてくれる仕組みがあるので、サイトをぼーっと見るだけでも勉強になります。

使用技術

HTML React (Vite) Three.js / react-globe.gl D3.js (d3-array) TopoJSON (topojson-client) World Atlas Pure CSS

アーキテクチャ

BoldChinese

プロジェクトについて

中国語の発音を録音すると、一漢字ごとに発音を返してくれます。

背景・課題

中国語は発音形態がかなり異なるほか、声調があるので自己学習が極めて難しいとされています。

主要機能

録音後フィードバック

録音から20秒ほどで、声調、発音両面からのフィードバックが行われます。

ダッシュボード

ダッシュボードから、どの漢字が苦手か、同じ文章でどのようなミスが行われているかが確認できます。

技術スタック

使用技術

Dart/Flutter Python/FastAPI Paddlespeech

プロジェクト成果

自分自身の発音の傾向確認

どの発音も実際苦手だと思っていましたが、実際は比較的得意な声調があることを、アプリ側からも知れた。

LLMによる小説翻訳差

プロジェクトについて

ChatGPT・Gemini・DeepSeekによる中国語翻訳結果を収集し、語彙や表現の違いを比較可視化した分析プロジェクトです。

背景・課題

複数モデルの翻訳スタイル差異を学習に活かすため、客観的にキャラを把握したいという課題がありました。

主要機能

語彙比較

モデルごとに頻出語を抽出し、語彙のニュアンス差を俯瞰できます。

構文パターン分析

係り受けやコロケーションを比較し、翻訳の特徴を定量的に把握します。

使用技術

Python 3 Pandas Matplotlib
